# Скрипт генерации вакансий

## Описание
Скрипт `generate_vacancies.py` генерирует и добавляет 300000 примерных вакансий через POST запрос к `/vacancy_create`.

## Установка зависимостей

```bash
pip install aiohttp
```

## Использование

1. Убедитесь, что сервер запущен на `http://localhost:8000` (или измените `BASE_URL` в скрипте)

2. Запустите скрипт:
```bash
python scripts/generate_vacancies.py
```

## Параметры конфигурации

В начале файла `generate_vacancies.py` можно изменить:
- `BASE_URL` - URL сервера (по умолчанию: `http://localhost:8000`)
- `BATCH_SIZE` - количество вакансий в одном запросе (по умолчанию: 100)
- `TOTAL_VACANCIES` - общее количество вакансий для генерации (по умолчанию: 300000)
- `CONCURRENT_REQUESTS` - количество параллельных запросов (по умолчанию: 10)

## Примечания

- Скрипт использует асинхронные запросы для ускорения процесса
- Вакансии генерируются с случайными данными из предопределенных списков
- Прогресс отображается в консоли
- При ошибках скрипт продолжает работу и выводит статистику в конце
