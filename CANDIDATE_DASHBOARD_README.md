# Личный кабинет кандидата

## Обзор

Создан шаблон личного кабинета кандидата с использованием Bootstrap 5 Grid и Dark Tech стиля, соответствующего дизайну остального сайта.

## Структура файлов

```
app/
├── templates/
│   └── candidate/
│       └── dashboard.html          # Основной шаблон личного кабинета
└── static/
    └── css/
        └── cabinet.css             # Стили для кабинета кандидата
```

## Особенности реализации

### 1. Наследование базового шаблона
- Использует `{% extends "layouts/base.html" %}`
- Сохраняет общую структуру сайта (sidebar, header)
- Не нарушает существующий дизайн

### 2. Префикс классов
- Все классы используют префикс `candidate-` для избежания конфликтов
- Примеры: `candidate-dashboard`, `candidate-card`, `candidate-profile-card`
- Не конфликтует с существующими классами типа `.profile-card`, `.sidebar`

### 3. Bootstrap 5 Grid
- Использует стандартную сетку Bootstrap 5
- Адаптивный дизайн: `col-lg-4` и `col-lg-8` для десктопа
- Автоматическое переключение на мобильную версию

### 4. Dark Tech стиль
- Использует CSS переменные из `globals.css`
- Золотые акценты (`--color-accent-gold`)
- Темный фон с эффектами свечения
- Соответствует общему стилю сайта

## Структура страницы

### Header Section
- Заголовок "Личный кабинет"
- Кнопка "Редактировать профиль"

### Left Column (col-lg-4)
1. **Profile Card** - Карточка профиля
   - Аватар
   - Имя и роль
   - Бейджи (Grade, Status)
   - Контактная информация (Email, Telegram, Локация, Опыт)

2. **Resume Card** - Блок резюме (Placeholder)
   - Иконка плейсхолдера
   - Кнопка "Загрузить резюме"

### Right Column (col-lg-8)
1. **Stats Cards** - Статистика
   - Активные отклики
   - Интервью
   - В ожидании

2. **Messages Card** - Блок сообщений (Placeholder)
   - Плейсхолдер для сообщений от рекрутеров
   - Ссылка "Все сообщения"

3. **Applications Card** - Блок откликов (Placeholder)
   - Плейсхолдер для активных откликов
   - Кнопка "Найти вакансии"

## Плейсхолдеры

Все блоки, которые будут наполнены позже, имеют класс `candidate-placeholder`:
- Блок резюме
- Блок сообщений
- Блок откликов

Плейсхолдеры содержат:
- Иконку
- Текст описания
- Кнопки действий (опционально)

## CSS классы

### Основные классы
- `.candidate-dashboard` - Контейнер страницы
- `.candidate-card` - Базовая карточка
- `.candidate-card-header` - Заголовок карточки
- `.candidate-card-body` - Тело карточки

### Специализированные классы
- `.candidate-profile-card` - Карточка профиля
- `.candidate-resume-card` - Карточка резюме
- `.candidate-messages-card` - Карточка сообщений
- `.candidate-applications-card` - Карточка откликов
- `.candidate-stat-card` - Карточка статистики

### Компоненты
- `.candidate-avatar` - Аватар пользователя
- `.candidate-badge` - Бейджи (Grade, Status)
- `.candidate-info-item` - Элемент информации
- `.candidate-placeholder` - Плейсхолдер блоков

## Адаптивность

### Desktop (≥992px)
- Двухколоночная сетка (4/8)
- Полная функциональность

### Tablet (768px - 991px)
- Одноколоночная сетка
- Вертикальное расположение блоков

### Mobile (<768px)
- Компактные отступы
- Уменьшенные размеры элементов
- Вертикальное расположение статистики

## Использование

### В роутере FastAPI
```python
@router.get("/candidate/dashboard")
async def candidate_dashboard(request: Request):
    return templates.TemplateResponse(
        "candidate/dashboard.html",
        {"request": request}
    )
```

### Передача данных
Шаблон готов к получению данных через контекст:
- `candidate_name` - Имя кандидата
- `candidate_email` - Email
- `candidate_role` - Роль/позиция
- `stats` - Статистика (отклики, интервью и т.д.)
